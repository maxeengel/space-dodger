Knappguide

Magicsee R1

  • Joystick Flytt ringen
  • Rund knapp foran Pause og fortsett
  • A / andre knapper Prøv igjen ved game over
  • Joystick + knapp Start spill fra meny

På skjermen

  • Pause Pause og fortsett (musikk fortsetter)
  • Musikk på/av Slå bakgrunnsmusikk av eller på

Tastatur

  • WASD / piltaster Flytt ringen
  • Mellomrom / Enter Start og prøv igjen
  • P / Esc Pause og fortsett

Spillmål

  • Samle gule energikuler (+10 poeng)
  • Unngå asteroider (3 liv)

R1 må være i spillmodus (slå av, hold M+B, slå på). Hvis volum endres på Mac, start R1 på nytt i spillmodus.
